# Motion Detection using OpenCV

This project detects moving objects using a webcam and OpenCV.

## Features

- Real-time motion detection
- Status text: Normal / Moving Object Detected
- Bounding box on moving objects
- Terminal output for motion status

## Technologies

- Python
- OpenCV
- Imutils

## Run the Project

```bash
pip install -r requirements.txt
python objectDetection.py
```
